Welcome to Gober API

API Documentation

Gober API is a robust and scalable backend RESTful API built using FastAPI. The core of this project is to manage users, bids (tenders), and custom search criteria while orchestrating long-running asynchronous tasks such as generating summaries and documents using AI.

Features

  • User authentication and management
  • Tender listing, filtering, and detailed view
  • Search criteria management
  • AI-powered document and summary generation
  • Asynchronous task processing
